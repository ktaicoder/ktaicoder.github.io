# PC 프로그램에서 오토런 설정하기

<MdxTextBox>
    <Box sx={{color:'secondary.main', fontWeight:600, mb:1}}>
        PC 프로그램에서 코딩팩의 오토런을 설정할 수 있습니다.
    </Box>
    <ul>
        <li>오토런은 2021년 12월 20일에 추가된 기능입니다. 최신 버전의 코딩팩 OS 이미지와 PC 프로그램을 설치해야 합니다.</li>
        <li>2021년 9월, 코딩블록 사이트의 대규모 리뉴얼 시점부터 오토런 기능을 제공하지 못하다가 12월 20일부터 다시 제공하기 시작했습니다.</li>
    </ul>
</MdxTextBox>

- 오토런은 코딩팩의 `노랑 버튼`을 터치했을 때 `자동으로 작품이 실행`되는 기능입니다.
- 오토런은 기존의 방식을 완전히 변경했습니다. 
- 기존에는 코딩팩에 GEN 파일을 저장하는 방식이었는데, 이제부터는 작품의 주소를 저장하는 방식으로 변경되었습니다.
- 기존에는 코딩팩 부팅시 자동 실행기능이 있었지만, 내부 논의 끝에 없애기로 결정했습니다.
    - 부팅시 자동 실행은 PC 프로그램의 사용을 불편하게 합니다.
- 기존의 오토런은 부팅 후 1회만 실행할 수 있었는데, 이제부터는 노랑 버튼을 터치하면, 언제든지 오토런이 실행됩니다.

오토런은 두 가지 방법으로 설정할 수 있습니다. 
- PC 프로그램을 이용해서 설정하거나 
- 코딩팩에서 코딩블록을 실행하여 `나의 작품` 페이지에서 등록할 수 있습니다.

## 목차
1. PC 프로그램에서 설정하기
1. 코딩팩에서 설정하기


## 1. PC 프로그램에서 설정하기

- PC 프로그램에서 오토런을 설정하는 것은 코딩팩에 모니터/키보드를 연결할 필요가 없으므로 편리합니다.
- PC 프로그램에서 코딩팩을 설정하는 것이므로 우선 코딩팩이 콘솔모드가 되어야 합니다.
- `콘솔 모드`에 대해 잘 모르신다면, <PostLink postId="hw-pc-how-to-codingpack-setup">코딩팩 설정하기</PostLink> 문서를 읽어주세요.
    - 문서를 요약하자면, 노랑 버튼을 4초간 누르면 코딩팩이 `콘솔 모드`가 되어 PC 프로그램에서 연결할 수 있다는 설명입니다.

### 오토런 메뉴 선택

<MdxImageView src="/images/hw-pc/codingpack_autorun1.png" caption="PC 프로그램의 오토런 메뉴"/>

- 오토런 메뉴를 클릭하면 오토런 주소를 입력하는 화면이 표시됩니다.

<MdxImageView src="/images/hw-pc/codingpack_autorun2.png" caption="오토런 주소 붙여넣기"/>

- 오토런 주소는 작품을 자동으로 실행할 수 있는 주소이며, 코딩블록 웹사이트에서 복사해와야 합니다.
- PC의 브라우저를 실행하여 코딩블록 웹사이트에서 `나의 작품` 페이지를 열어주세요.
- `나의 작품`은 코딩팩에서 접속할 때와 PC에서 접속한 경우의 사용법이 조금 다릅니다.
- 먼저 PC에서 접속해 봅니다.

<MdxImageView src="/images/hw-pc/codingpack_autorun3.png" caption="코딩블록 사이트의 나의 작품"/>

- `나의 작품` 화면에 있는 `오토런 버튼`을 클릭하면 `작품 등록 팝업`이 나옵니다.
- 팝업에서 `오토런 작품 등록` 버튼을 클릭하면 `오토런 주소`가 나옵니다. 이 주소를 복사해주세요.

<MdxImageView src="/images/hw-pc/codingpack_autorun4.png" caption="오토런 주소 생성"/>

- 위에서 복사한 주소를 PC 프로그램에 붙여넣기 해주세요.

<MdxImageView src="/images/hw-pc/codingpack_autorun5.png" caption="오토런 주소 붙여넣기"/>

- 코딩팩의 노랑 버튼을 눌렀다 떼면 방금 등록한 오토런이 실행됩니다.

<MdxImageView src="/images/hw-pc/codingpack_autorun6.png" caption="노랑 버튼을 살짝 눌렀다 떼기"/>

- 2초 정도 기다리면 아래와 같이 자동으로 브라우저가 실행되고, 작품이 실행됩니다.

<MdxImageView src="/images/hw-pc/codingpack_autorun7.png" caption="오토런이 실행된 상태"/>

## 2. 코딩팩에서 설정하기
<MdxTextBox>
    <Box sx={{color:'secondary.main', fontWeight:600, mb:1}}>
        코딩팩에서는 PC 프로그램 없이 오토런을 설정할 수 있습니다.
    </Box>
</MdxTextBox>

- 코딩팩의 바탕화면에서 `AI 블록 코딩 실행하기`를 더블 클릭하면 브라우저가 실행됩니다.
- 로그인 후에 `나의 작품` 페이지에서 오토런 버튼을 클릭하면 오토런 등록 팝업이 표시됩니다.
- 코딩팩에서는 PC에서처럼 주소를 복사할 필요가 없습니다. `오토런 작품 등록` 버튼을 클릭하면 코딩팩에 즉시 저장되기 때문입니다.
- 아래는 코딩팩과 PC에서 실행할 때를 비교하는 화면입니다.

<MdxImageView src="/images/hw-pc/codingpack_autorun8.png" caption="코딩팩과 PC에서의 오토런이 생성 비교"/>

- 오토런을 실행하려면, 현재 실행중인 블록코딩을 종료해야 합니다. 노랑 버튼을 `4초간` 누르고 있으면 블록코딩이 종료됩니다.
- 블록코딩이 종료된 상태에서 코딩팩의 노랑 버튼을 살짝 눌렀다 떼면 오토런이 실행됩니다.
