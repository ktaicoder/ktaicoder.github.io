## 통합개발환경 기능

-   신택스 하이라이팅
    -   IDE는 구문 강조 표시를 제공하여 코드를 쉽게 읽고 쓸 수 있도록 합니다. 코드의 다른 부분은 빠르게 식별할 수 있도록 다양한 색상으로 표시됩니다.
-   시리얼 모니터
    -   직렬 모니터를 사용하면 아두이노보드와 통신하고 실시간으로 아두이노보드가 무엇을 하는지 볼 수 있습니다. 이 기능은 코드를 디버깅하고 문제를 해결하는 데 유용합니다.
-   기본 제공 라이브러리
    -   통합개발환경에는 센서, 디스플레이 및 기타 하드웨어 구성 요소와 인터페이스하는 데 사용할 수 있는 다양한 기본 제공 라이브러리가 함께 제공됩니다.
-   자동 완성
    -   통합개발환경에는 입력 시 코드 완료를 제안하는 자동 완성 기능이 포함되어 있습니다. 이렇게 하면 코드를 더 빠르고 정확하게 작성할 수 있습니다.
-   업로드 매니저 - 통합개발환경에는 코드를 아두이노보드에 쉽게 업로드할 수 있는 업로드 관리자가 포함되어 있습니다. 업로드 프로세스에 대한 피드백도 제공하여 오류나 문제가 있었는지 확인할 수 있습니다.

## 기본 프로그램 구조

아두이노 프로그램은 크게 두 부분으로 나눌 수 있습니다:

-   Setup 함수는 프로그램이 시작될 때 한 번만 실행됩니다. 이 기능은 일반적으로 초기화에 사용됩니다.
-   Loop 함수는 설정 기능 이후에 반복적으로 실행되며 프로그램의 주요 내용을 포함합니다.
