### 복구하는 방법

- 복구하는 과정은 <PostLink postId="codingpack-how-to-open-terminal">터미널</PostLink>에서 이루어지는 것이라 조금 성가신 작업입니다. 
- 개발자들에겐 익숙하지만, 일반 사용자들에겐 조금 어려울 수 있습니다.

<PostLink postId="codingpack-how-to-open-terminal">터미널</PostLink>을 열고 아래의 명령을 실행하면 됩니다. 

#### 복구 명령어 실행

```bash
curl https://raw.githubusercontent.com/aicodingblock/reset_blockdriver/release/blockDriver/rescue.sh | bash
```

위의 명령을 복사, 붙여넣기, 실행(엔터) 하면 됩니다. 아래와 같이 붙여넣기 해주세요.
<MdxImageView src="/images/doc/rasp_system_rescue1.png" caption="복구 명령어 입력"/>

##### TIP! 터미널에서 복사와 붙여넣기 단축기

보통 복사할때 `CTRL + C`, `CTRL + V` 하시죠? 하지만, <PostLink postId="codingpack-how-to-open-terminal">터미널</PostLink>에서는 이 단축키를 이용할 수 없습니다.
- 터미널에서 복사는 `CTRL + INSERT`, 붙여넣기는 `SHIFT + INSERT` 입니다.
- 참고로 터미널에서 `CTRL + C`는 취소(cancel)입니다. 실행중인 명령을 중지할 때 사용합니다.
