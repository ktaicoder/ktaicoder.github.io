전원 입력

아두이노 보드에 전원을 공급하는 방법은 여러 가지가 있지만

  • 가장 일반적인 방법은 컴퓨터에 연결된 USB 케이블 또는 USB 전원 어댑터를 사용하여 USB 포트를 통해 전원을 공급하는 것입니다.
  • 또 다른 옵션은 전원 공급 장치를 보드의 전원 잭에 연결하는 것입니다.
  • 전원 잭은 7~12V의 전압 범위를 허용합니다.
  • 또한 일부 아두이노보드에는 배터리 또는 벽면 어댑터와 같은 외부 전원을 사용하여 보드에 전원을 공급하는 데 사용할 수 있는 Vin 핀이 있습니다.

전원 출력

아두이노에서 출력을 생성하려면 GND 핀과 함께 보드의 레귤레이터를 사용하면 됩니다.

LED

LED는 전기가 흐를 때 빛을 내는 작은 전자 부품입니다.

LED

  • 아두이노 우노와 함께 LED를 사용하려면 LED를 통과하는 전류량을 제한하기 위해 저항을 사용하여 보드의 디지털 핀 중 하나에 LED를 연결해야 합니다.

  • LED의 긴 다리는 양극(+)이고, 짧은 다라는 음극(-)입니다.

  • 양극은 디지털 핀에 연결하고 음극은 저항기를 통해 접지에 연결해야 합니다.

LED 사용방법

LED가 연결되면 아두이노 통합 개발 환경를 사용하여 디지털 핀을 high, low로 설정하여 LED를 켜고 끄는 프로그램을 작성할 수 있습니다.

아래는 매 초마다 LED를 켜고 끄는 간단한 코드입니다.

int ledPin = 13; // LED connected to digital pin 13

void setup() {
    pinMode(ledPin, OUTPUT); // sets the digital pin as output
}

void loop() {
    digitalWrite(ledPin, HIGH); // turns on the LED
    delay(1000); // waits for a second
    digitalWrite(ledPin, LOW); // turns off the LED
    delay(1000); // waits for a second
}

브레드 보드

브레드 보드는 다양한 설계를 실험하고 테스트하는 데 이상적인 도구입니다.

브레드 보드

  • 브레드 보드는 시제품 제작과 실험을 위한 임시 전자 회로를 만드는 데 사용되는 장치입니다.

  • 아두이노 우노와 관련해서는 아두이노 보드에 저항기, LED, 센서 등 다양한 전자부품을 납땜하지 않고도 연결할 수 있는 브레드보드를 사용할 수 있습니다.

브레드보드 사용방법

우노 와 함께 브레드보드를 사용하려면 다음 단계를 수행합니다:

  • 브레드보드를 평평한 표면에 놓고 방향이 올바른지 확인합니다.
  • 아두이노 우노의 핀을 브레드보드의 해당 행에 삽입하여 브레드보드에 연결합니다.
  • 사용할 아두이노의 핀을 식별하고 해당 핀에 점퍼선을 삽입합니다.
  • 점퍼 와이어의 다른 쪽 끝을 브레드보드의 해당 행에 연결합니다.
  • LED 또는 저항기와 같은 전자 구성 요소를 브레드보드에 삽입하고 적절한 점퍼 와이어에 연결합니다.
  • 전자 부품을 제어하는 코드를 작성하여 보드에 업로드합니다.
  • 전원 공급 장치를 켜고 회로가 의도한 대로 작동하는지 테스트합니다.

저항

저항
  • 저항은 아두이노 우노를 사용하는 상황을 포함하여 전자 공학에서 중요한 구성 요소입니다.
  • 종종 전기의 흐름을 제어하기 위해 LED와 센서와 같은 다른 구성 요소와 함께 사용됩니다.

저항 사용방법

  • 아두이노와 함께 저항기를 사용하려면 일반적으로 LED 또는 사용하려는 다른 구성 요소와 직렬로 연결해야 합니다.
  • 이는 회로를 통과하는 전류의 양을 제한하기 위해 사용되며, LED 또는 다른 구성 요소가 연소되는 것을 방지할 수 있습니다.
  • 사용하는 특정 저항 값은 전원 공급기의 전압과 회로에 필요한 전류에 따라 달라집니다.
  • 저항을 선택할 때 저항의 저항 값, 정격전력, 공차를 고려해야 합니다.
  • 저항 값은 저항기가 전류 흐름을 제한하는 정도를 결정하고
  • 정격전력은 저항기가 과열 없이 견딜 수 있는 소비전력으로, 전류의 제곱x저항으로 계산할 수 있습니다.
  • 정격전력보다 큰 전류를 인가하면, 저항기에 열이 발생하여 타버리는 경우도 있습니다.
  • 공차는 저항기의 실제 저항이 명시된 값과 얼마나 다를 수 있는지를 지정합니다.
