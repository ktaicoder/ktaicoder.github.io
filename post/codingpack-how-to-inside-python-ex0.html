코디니팩의 파이썬 예제 실행 준비 과정

이 문서는 코디니팩 라즈베리파이의 파이썬 예제를 실행하기 위한 준비 과정에 대한 설명입니다.

  • 이 예제는 2022년 11월 28일부터 제공하기 시작한 새로운 파이썬 예제입니다.
  • 코딩팩 바탕화면의 시스템 초기화를 클릭하면, 새로운 파이썬 예제가 추가됩니다.

  • 새로운 파이썬 예제는 사용자 인증(user_auth.py) 부분을 개선하여, API 키를 발급받아야 했던 기존의 과정이 필요없습니다.

  • 사용자 인증 부분 외에는 기존의 예제(ai-markers-kit/python3 폴더)와 동일합니다.

  • 새로운 파이썬 예제는 새로운 인공지능 서버에 연결됩니다. 기존의 예제는 오래된 서버에 연결하는데, 2022년 12월 31일 부로 서비스가 종료됩니다.
