# AI 코딩블록 개발자 가이드

AI 코딩블록 개발자 가이드 웹사이트 프로젝트입니다.

https://ktaicoder.github.io


## 다음과 같이 설치할 수 있습니다.

```bash
$  cd ./root
$  yarn install
$  yarn dev
```

브라우저에서 http://localhost:3000 에 접속합니다.
